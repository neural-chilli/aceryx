# Batch-Close Checklist

1. Reality Matrix updated (implemented vs planned vs stale claims)
2. Acceptance criteria verified with evidence
3. Security/tenant/RBAC regressions checked
4. Docs/spec diffs aligned to current behavior
5. Handoffs written (delta only)
6. Next batch scoped (3–6 tasks max, independent where possible)
7. Commit/PR process executed per normal workflow (unless a temporary session constraint says otherwise)
